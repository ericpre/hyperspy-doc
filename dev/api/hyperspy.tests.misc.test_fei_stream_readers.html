hyperspy.tests.misc.test_fei_stream_readers module¶

Test the FEI stream readers.

Because there is no official description of the format, these tests just tests consistency between array_to_stream and stream_to*array. In the particular case of stream to sparse array, we use dask to compute the array in order to mimic the usage in the FEI EMD reader.
